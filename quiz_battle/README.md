## 기능 소개

### 메인 기능

1. Socket 통신 기반 1:1 대결 매칭

   > Room 번호를 기반으로 Socket 통신을 활용하여 두 사용자를 매칭 <br/>
   > 실시간 이벤트를 통해 문제 진행 상황 및 결과가 즉시 동기화됨 <br/>

2. 매칭 대기 시스템

   > 사용자가 게임 시작을 요청하면 매칭 대기열에 등록 <br/>
   > 다른 플레이어가 접속하여 동일한 방에 입장할 때까지 대기 화면 유지 <br/>
   > 매칭이 완료되면 자동으로 게임이 시작됨 <br/>

3. 게임 중 이탈 감지 및 상태 표시

   > 게임 도중 상대방이 나갔을 경우 이를 실시간으로 감지 <br/>
   > 상대방이 나갔을 때 UI에 이탈 상태를 표시 <br/>

4. 결과 화면 제공

   > 게임 종료 후 승패 여부를 표시하는 결과 화면 제공 <br/>
   > 각 플레이어의 정답 수, 남은 HP 등의 정보 표시 <br/>
   > 결과에 따른 간단한 반응형 UI 제공 <br/>

5. Multi Room 지원

   > 여러 게임 방을 동시에 생성 및 관리 가능 <br/>
   > 각 방은 독립적으로 게임이 진행됨 <br/>
   > 서버에서 방 단위로 Socket 연결을 관리 <br/>

6. JSON 기반 정적 DB를 활용한 랜덤 문제 출제

   > 문제 데이터는 JSON 형태의 정적 데이터베이스로 관리 <br/>
   > 게임 시작 시 서버에서 랜덤으로 문제를 선택하여 출제 <br/>
   > 동일한 문제 세트를 두 플레이어에게 동시에 제공하여 공정한 게임 진행 보장 <br/>

### 부가 기능

1. HP Bar 감소 애니메이션

   > 문제를 맞출 경우 상대방에게 데미지를 주는 방식 <br/>
   > HP가 감소할 때 애니메이션 효과로 시각적으로 표현 <br/>
   > 실시간으로 두 플레이어의 상태 확인 가능 <br/>

## 실행 방법

1. VScode에 git 설치
2. node.js 설치
3. npm -v, node -v를 통해 설치확인
4. VScode 다시로드
5. 명령어 실행

```bash
npm install
node server.js
```

접속(로컬): `http://localhost:3000/quiz` <br/>
접속(외부): `http://IPv4주소:3000/quiz`
